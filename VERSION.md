# 版本记录

记录日期：2026-06-24

## 卢卡斯热词1号

“卢卡斯热词1号”指最初的 B站关键词热度排查工具版本。

包含能力：

- 本地 Web 可视化界面
- B站关键词视频搜索采集
- B站搜索框联想词采集
- 关键词热度、增长、竞争和机会分汇总
- 明细、汇总、联想词 CSV 下载
- 默认使用 `curl` + IPv4 请求 B站
- 匿名 B站 Cookie 会话预热和 `-412 request was banned` 后的 Cookie 刷新重试
- `restart_app.bat` 本地重启脚本

以后提到“卢卡斯热词1号”，即指以上能力集合对应的版本。

## 卢卡斯热词2号

“卢卡斯热词2号”基于 1 号迭代，新增选题级分析能力。

新增能力：

- 输入一个完整选题，而不只是关键词
- 自动拆解选题中的重要关键词
- 对拆解关键词继续调用 B站真实采集能力
- 计算整个选题的综合评分
- 给出 `可执行`、`小样本测试` 或 `暂不建议` 的选题建议
- 自动生成多条优化选题，并为每条优化选题打分
- 新增选题总分、拆解关键词、优化选题 CSV 下载
- 保留 1 号原有关键词排查模式

2 号的选题总分主要由这些维度组成：

```text
topic_score =
45% 关键词机会分
+ 20% 搜索意图清晰度
+ 15% 选题表达清晰度
+ 10% 差异化空间
+ 10% 数据可信度
- 风险扣分
```

其中关键词机会分继续沿用 1 号的真实 B站数据评分体系。
